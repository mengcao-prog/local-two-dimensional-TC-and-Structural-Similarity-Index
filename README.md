# Overview

This repository contains MATLAB code for implementing and evaluating the TC_2D_local merging framework and SSIM-based spatial evaluation metrics for soil moisture products.

  The project focuses on:
  
      Local two dimensional Triple Collocation (TC_2D_local)
      
      SSIM-based spatial evaluation

# TC_2D_local

    Local spatial covariance estimation using moving windows
    
    Combination of temporal and spatial TC information
    
    Adaptive weighting based on valid spatial and temporal samples
    
    Flexible TC window size configuration


# SSIM Evaluation

    Implementation of: SIM, SIV, SIP, SSIM
    
    with configurable SSIM window sizes.
